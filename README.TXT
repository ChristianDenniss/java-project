Shut The Door - JavaFX Number Game

🏆 About the Game

Shut The Door is an engaging number game built with JavaFX, designed for streamers to play with a randomly selected viewer. The game combines strategy and luck, ensuring an interactive and exciting experience for both the streamer and their audience.

🎮 Features

JavaFX UI for an intuitive and visually appealing experience.

Streamer & Viewer Interaction: The streamer plays against a randomly chosen viewer.

Turn-Based Gameplay with easy-to-use controls.

Randomized Number Generation to keep each round unpredictable.

Win/Loss Conditions that define clear game outcomes.

Modular Code Structure for easier maintenance and expansion.

🔧 Installation & Setup

Prerequisites

Ensure you have the following installed:

Java (JDK 17 or later recommended)

JavaFX SDK

An IDE like IntelliJ IDEA or Eclipse (with JavaFX support configured)

📥 Clone the Repository

git clone https://github.com/YourRepo/ShutTheDoor.git
cd ShutTheDoor

▶️ Running the Application

Ensure JavaFX libraries are correctly configured in your IDE.

Compile and run the main application:

javac --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml Main.java
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml Main

The game window should launch, allowing the streamer to play with a selected viewer.

🎲 How to Play

The game starts with a set of numbered tiles (or buttons).

The streamer and viewer take turns selecting numbers based on the game rules.

The objective is to "shut the door" by achieving a specific condition or blocking the opponent.

The game ends when a win/loss condition is met.
